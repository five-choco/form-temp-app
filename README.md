# フォーム入力アプリケーション (form-react-app)

このプロジェクトは、ReactとTypeScriptで構築された、基本的なバリデーション機能を持つ入力フォームアプリケーションです。Viteをビルドツールとして使用しています。

## 主な使用技術

- React
- TypeScript
- Vite

## 機能一覧

- **名前の入力:** 姓、名、セイ、メイをそれぞれ入力します。
- **性別の選択:** ラジオボタンで性別を選択します。
- **年齢の入力:** 年齢を入力します。
- **リアルタイムバリデーション:**
    - 名前フィールド: 記号と数字の使用を禁止します。
    - 年齢フィールド: 半角数字のみ入力を許可します。
    - エラーがある場合、フィールド下にエラーメッセージが表示されます。
- **送信ボタンの制御:** 全ての必須項目が正しく入力されるまで、「次へ」ボタンは無効化されます。

## セットアップと実行方法

### 1. 依存関係のインストール

プロジェクトルートで以下のコマンドを実行し、必要なパッケージをインストールします。

```bash
npm install
```

### 2. 開発サーバーの起動

以下のコマンドで開発サーバーを起動します。

```bash
npm run dev
```

サーバーが起動すると、ブラウザでアプリケーションにアクセスできるようになります。

### 3. ビルド

本番環境向けのファイルを生成するには、以下のコマンドを実行します。

```bash
npm run build
```

ビルド成果物は `dist` ディレクトリに出力されます。

### 4. Linterの実行

コードの静的解析を実行するには、以下のコマンドを使用します。

```bash
npm run lint
